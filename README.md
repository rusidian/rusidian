## About Me

저는 광운대학교 정보융합학부를 졸업했으며, 소프트웨어 개발과 데이터 분석을 중심으로 한 전공 기반 역량을 갖추고 있습니다.

Python 프로그래밍과 알고리즘 문제 해결 경험을 바탕으로 데이터 처리 및 분석을 수행해왔으며, 시각화 및 대시보드 프로젝트를 통해 데이터 기반 인사이트를 도출하는 작업을 진행했습니다.

학부 과정 동안 산학협력 프로젝트와 졸업작품전시회에 참여하여 소프트웨어 프로젝트 성과로 수상한 경험이 있습니다.

현재는 Software Engineering 및 Data 관련 직무로의 진출을 목표로, 기술 역량과 프로젝트 포트폴리오를 지속적으로 강화하고 있습니다.

### ESG Sentiment Analysis using LLM  
https://github.com/rusidian/LLM_ESG_POS

- ESG 관련 텍스트 데이터를 활용한 감성 분석 프로젝트
- LLM 기반 자연어 처리 모델을 활용하여 긍·부정 분류 수행
- 데이터 수집 및 전처리 파이프라인 구축
- 텍스트 정제 및 토큰화 처리 진행
- 분석 결과 기반 인사이트 도출

기업 및 사회 이슈에 대한 감성 흐름을 분석한 프로젝트입니다.  
대규모 언어모델(LLM)을 활용하여 텍스트 분류 성능을 개선하고,  
데이터 전처리부터 분석까지 전체 파이프라인을 구축했습니다.
